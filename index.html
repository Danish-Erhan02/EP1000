What is about this website

This website is a documentation of my progress of the Digital Fabrication (FabLab) Elective Module in Singapore Polytechnic.

As the name suggests, FabLab is a small scale workshop offering digital fabrication. A fab lab is typically equipped with an array of flexible computer-controlled tools that cover several different length scales and various materials, with the aim to make "almost anything".
